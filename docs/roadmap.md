# type-globe ロードマップ

## 設計の方針転換（2026-04-30）

v0.1.x の「画面に出た文字列を打つタイピングモード」を **廃止**。コア原理を以下に再定義した：

> **「打つべき文字列が画面に出ないタイピングゲーム」**

- クイズ提示 × ランキング系（クイズ / Time Attack 25）
- リスニング提示 × ハクスラ RPG
- 両軸は**クロスさせない**

詳細は [spec.md](./spec.md) と [overview.md](./overview.md) を参照。

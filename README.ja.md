# type-globe

[English](README.md)

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

ターミナルで遊べるクイズ＆タイピングゲーム。知識を増やしながらタイピングも速くなる、一石二鳥のCLIゲームです。

## なぜ type-globe？

- **遊びながら学べる** -- プログラミング、アニメ、科学、歴史など100問のクイズを収録
- **タイピング速度を測定** -- リアルタイムでWPMと正確性を表示
- **ステルスモード**（開発中） -- ターミナルに溶け込む見た目。仕事中でもバレません
- **オフライン動作** -- インターネット不要。シングルバイナリで完結
- **日英バイリンガル** -- 日本語・英語の両方に対応

## 特徴

- 4択クイズモード（スコア・正答率の追跡付き）
- タイピングモード（WPM・正確性・進捗をリアルタイム表示）
- [ratatui](https://github.com/ratatui/ratatui) によるTUI -- どのターミナルでも動作
- 15ジャンル・100問の厳選クイズ
- 起動時に言語選択（日本語 / English）
- シングルバイナリ、設定不要

## クイックスタート

```sh
cargo install type-globe
type-globe
```

## ゲームモード

| モード | 状態 | 説明 |
|--------|------|------|
| クイズ | 実装済み | 4択問題に回答。スコアと正答率を記録 |
| タイピング | 実装済み | 表示されたテキストを入力。WPMと正確性をリアルタイム計測 |
| クイズ+タイピング | 開発中 | クイズに答えた後、正解をタイプ |
| タイムアタック25 | 開発中 | アタック25風のパネル式クイズ |
| RPG（TypeQuest） | 開発中 | クイズ回答で経験値を獲得してレベルアップ |
| ステルス | 開発中 | ターミナルの作業に溶け込むモード |

### キーバインド

**メニュー**

| キー | 操作 |
|------|------|
| `↑` `↓` | 選択 |
| `Enter` | 決定 |
| `Esc` | 戻る |
| `q` | 終了 |

**クイズモード**

| キー | 操作 |
|------|------|
| `↑` `↓` | 選択肢を移動 |
| `1`-`4` | 選択肢に直接ジャンプ |
| `Enter` / `Space` | 回答 |
| `s` | スキップ |
| `q` | 終了 |

**タイピングモード**

| キー | 操作 |
|------|------|
| 任意の文字 | タイピング |
| `Backspace` | 削除 |
| `Esc` / `Ctrl+q` | 終了 |

## クイズジャンル

IT・プログラミング、Web開発、科学、数学、歴史、地理、言語、文化、アニメ、マンガ、ゲーム、VTuber・ネット文化、一般常識など。

## インストール

### crates.io から

```sh
cargo install type-globe
```

### ソースから

```sh
git clone https://github.com/kako-jun/type-globe.git
cd type-globe
cargo build --release
./target/release/type-globe
```

## 動作要件

- Unicode と基本的な ANSI エスケープコードに対応したターミナル
- Rust 1.70+（ソースからビルドする場合）

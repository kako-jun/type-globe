# type-globe

[English](README.md)

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

> **打つべき文字列が画面に出ない**タイピングゲーム。

百人一首の競技かるた（下の句は読まれず、上の句から記憶で札を取る）と同じ原理。**反射神経ではなく、知識・記憶・聴解で勝負する**タイピングゲームです。

## 仕組み

提示方式とゲーム構造は2軸あり、**クロスさせません**。

| 提示 | 組み合わせるモード | 鍛えるもの |
|---|---|---|
| **クイズ** — 質問と4択を見て、正解の文字列を打つ | 単発 / Time Attack 25 / **ランキング** | 知識 |
| **リスニング** — 音声で読まれる文を聞いて打つ | **ハクスラ RPG**（10問=1潜入） | 聴解 |

クイズモードでは**矢印キーでの選択はありません**。**正解の文字列を直接打ち始める**ことで選択を兼ねます。Enter で確定。

リスニングモードでは**画面に文字は一切出ません**。♪ の音符が `jiwa` 風に脈動しているだけ。聞いた音をブラインドで打ちます。

## なぜ type-globe？

- **反射タイピングではない** — 普通のタイピングゲームは画面の文字を打つだけ。本作は違います
- **短時間プレイ** — 1回の RPG ランは10問固定。ローグライク的「1回潜って帰る」サイクル
- **オフライン動作** — シングルバイナリ、ネットワーク不要
- **多言語** — 日本語・英語に対応（TTS が他言語も処理）
- **問題文のアニメーション表示（jiwa）** — クイズ文は1文字ずつフェードインで現れます。知っている人は出待ちせず先打ちできます

## 表示ルール

**禁則**（コア原理を破る）:
- 答えの文字列を入力前に表示すること
- 演出のスキップ

**許容**:
- 質問文・選択肢ラベル（クイズのみ。リスニングは無し）
- ユーザーが**実際に打った**文字（タイポ修正のため）
- ステータス類（HP / Lv / EXP / 残時間 / CPM / WPM）

## スコア

**CPM**（1分あたり文字数）と **WPM**（1分あたり単語数）を併記表示。

## 音声

リスニング音声は [`tts`](https://crates.io/crates/tts) crate で OS ネイティブ TTS を実行時に呼び出します（Linux: speech-dispatcher、macOS: AVSpeechSynthesizer、Windows: SAPI）。事前録音ファイルは同梱しません。リプレイは**無制限・ペナルティなし**（タイム消費が自然なペナルティ）。

## キーバインド

**クイズ**

| キー | 操作 |
|---|---|
| 文字キー | 正解の文字列を打つ |
| `Enter` | 確定 |
| `Tab` | スキップ |
| `F5` | リスタート |
| `Esc` | 終了 |

**リスニング RPG**

| キー | 操作 |
|---|---|
| 文字キー | 聞いた文字列を打つ |
| `Enter` | 確定 |
| `Space` | 音声リプレイ（無制限） |
| `F5` | 新しいラン |
| `Esc` | 町へ戻る |

## インストール

### crates.io から

```sh
cargo install type-globe
```

### ソースから

```sh
git clone https://github.com/kako-jun/type-globe.git
cd type-globe
cargo build --release
./target/release/type-globe
```

## 動作要件

- Unicode と ANSI エスケープに対応したターミナル
- OS の TTS バックエンド（最近の macOS / Windows / 主要 Linux ディストロは標準で動作）
- Rust 1.70+（ソースビルド時）
